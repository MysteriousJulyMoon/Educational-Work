## 🚀 Обзор некоторых проектов

### **Лабораторная работа 4: Графика с SFML**
**Папка:** `Labs 4-th/`

Графическое приложение на C++ с использованием библиотеки SFML (Simple and Fast Multimedia Library). Проект демонстрирует создание и отрисовку различных геометрических фигур.
